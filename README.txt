Это телеграмм бот для Создание викторин.

Чтобы редактировать вопросы нужно обратится к файлу qvest.py
Можно добавить вопросы скопировав:
{
        "question": " (Номер вопроса и сам вопрос вписать заместо этого текста) ",
        "options": ["Варианты ответов", "Варианты ответов", "Варианты ответов"],
        "correct_option": 1 # (  правильный ответ, от 0 "первый вариант"* до 2 )
    },

    Можно добавлять или убавлять необходимое колличество вопросов по вашему умотрению.

После необходимо отредактировать в том же файле coll, сколько баллов можно набрать (писать столько сколько вопросов)*


Чтобы подключить своего бота, необходимо вести токен вашего бота в API.json

Чтобы отредактирвать приветствие (/start), просто отредактируйте qvest.py, строчку start_on.
